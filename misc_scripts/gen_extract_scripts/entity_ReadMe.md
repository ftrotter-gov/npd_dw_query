Entity Research Readme
================

For the Medicare Enrollment Data (From PECOS by stored in the mdcr_prvdr section of IDR) The main entity lookup table is 

V2_MDCR_PRVDR_ENRLMT (or with history: V2_MDCR_PRVDR_ENRLMT_HSTRY)

With the following linkage: 

CREATE TABLE IF NOT EXISTS IDRC_PRD.CMS_VDM_VIEW_MDCR_PRD.V2_MDCR_PRVDR_ENRLMT (
    PRVDR_ENRLMT_ID TEXT(15) NOT NULL,
    -- PRVDR_ENRLMT_ID: PECOS SYSTEM GENERATED ENROLLMENT IDENTIFIER.
    -- Categories: enrollment
    PRVDR_PAC_ID TEXT(10) NOT NULL,
    -- PRVDR_PAC_ID: THE SYSTEM GENERATED PECOS ASSOCIATE CONTROL ID.
    -- Categories: pac_id
    PRVDR_ENRLMT_TIN_NUM TEXT(9) NOT NULL
    -- PRVDR_ENRLMT_TIN_NUM: TAX IDENTIFICATION NUMBER, SUCH AS AN EMPLOYER IDENTIFICATION NUMBER (EIN) OR SOCIAL SECURITY NUMBER...
    -- Categories: tin
);

The assignment record is: 

CREATE TABLE IF NOT EXISTS IDRC_PRD.CMS_VDM_VIEW_MDCR_PRD.V2_MDCR_PRVDR_MMBR (
    PRVDR_PAC_ID TEXT(10) NOT NULL,
    -- PRVDR_PAC_ID: THE SYSTEM GENERATED PECOS ASSOCIATE CONTROL ID.
    -- Categories: pac_id
    PRVDR_ENRLMT_ID TEXT(15) NOT NULL,
    -- PRVDR_ENRLMT_ID: PECOS SYSTEM GENERATED ENROLLMENT IDENTIFIER.
    -- Categories: enrollment
    PRVDR_INDVDL_ENRLMT_ID TEXT(15),
    -- PRVDR_INDVDL_ENRLMT_ID: ENROLLMENT ID OF INDIVIDUAL ENROLLMENT
    -- Categories: enrollment
    PRVDR_GRP_MMBR_PAC_ID TEXT(10)
    -- PRVDR_GRP_MMBR_PAC_ID: PAC ID OF GROUP MEMBER
    -- Categories: pac_id
);
